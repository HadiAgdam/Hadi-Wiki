| Pattern | Meaning                  | Example Output |
| ------- | ------------------------ | -------------- |
| `yyyy`  | Year (4 digits)          | `2025`         |
| `yy`    | Year (2 digits)          | `25`           |
| `MM`    | Month (01-12)            | `06` (June)    |
| `MMM`   | Month name (short)       | `Jun`          |
| `MMMM`  | Month name (full)        | `June`         |
| `dd`    | Day of the month (01-31) | `09`           |
| `E`     | Day of the week (short)  | `Fri`          |
| `EEEE`  | Day of the week (full)   | `Friday`       |
| `HH`    | Hour (24-hour, 00-23)    | `14`           |
| `hh`    | Hour (12-hour, 01-12)    | `02`           |
| `mm`    | Minutes (00-59)          | `30`           |
| `ss`    | Seconds (00-59)          | `45`           |
| `a`     | AM/PM marker             | `PM`           |
| `z`     | Time zone (short)        | `GMT`          |
| `Z`     | Time zone offset         | `+0500`        |
